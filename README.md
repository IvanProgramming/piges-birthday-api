# Piges Birthday API

Simple API, that returns birthdays of minipigs

## Created with

- [Cloudflare Workers](https://workers.cloudflare.com/)
- [Cloudflare KV](https://workers.cloudflare.com/)
- [SheetsDB](https://sheetsdb.io/)
- [Hono](https://honojs.dev)
